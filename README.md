# 🐠 AquaAura Fish World

A modern and visually appealing **static fish shop website** built using HTML, CSS, and JavaScript.
This project showcases different types of aquarium fishes along with basic information, designed for a smooth and engaging user experience.

---

## 📌 About This Repository

This repository contains the complete source code for a **frontend-only static website** that can be easily deployed using GitHub Pages.

The website is designed to:

* Display various aquarium fish species 🐟
* Provide basic information about each fish
* Offer a clean, responsive, and attractive UI
* Serve as a beginner-friendly web project

---

## 🤖 Built with AI Assistance

This project was created with the help of **Artificial Intelligence tools** to explore:

## 🐟 About the Fish Shop

**AquaAura Fish World** is a conceptual aquarium store that focuses on:

* Breeding and maintaining high-quality aquarium fishes
* Promoting beginner-friendly pet fishes
* Encouraging responsible fish keeping

### 🌊 Fish Types Included:

* Guppy Fish
* Angel Fish
* Betta (Fighter) Fish
* Gold Fish
* Molly Fish
* Tetra Fish
* Iridescent Shark (Baby Shark)

Each fish is displayed with:

* Image 🖼️
* Name
* Short description

## 🚀 How to Run Locally

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/aqua-aura.git
   ```

2. Navigate into the project folder:

   ```bash
   cd aqua-aura
   ```

3. Open `index.html` in your browser

---

## 🌐 Deployment

This project can be easily deployed using **GitHub Pages**:

* Go to repository **Settings**
* Navigate to **Pages**
* Select branch → `main`
* Your site will be live 🎉
